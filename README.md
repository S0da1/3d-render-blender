What I am creating is a 3D engine for Blender running on Apple's M-series chips. I wanted to see how far I could create a realistic engine like Redshift or Octane ( i know it will never be as good clearly ), while 
having it optimised as much as possible and render as quick as possible on the latest Apple's chips. <br>
